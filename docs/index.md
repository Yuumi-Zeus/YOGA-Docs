# OdinToolkits

## 简介

`OdinToolkits` 是一个开源项目，个人开发插件，`Unity Odin Inspector` 插件的扩展工具集，旨在挖掘 `Odin Inspector` 插件用法，补充 `Unity` 编辑器开发。

同时内部包含一个模块化的游戏开发架构，基于 `ScriptableObject` 构建，类似于 `Soap` 插件。

如果觉得项目部分内容不错，并采用的话，请给个 `Star` 吧！
